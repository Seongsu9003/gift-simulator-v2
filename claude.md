# 📋 프로젝트 인수인계 지침서: 아빠의 '서른 살엔 독립해라' 계산기

**To. Claude (AI Developer)**
이 프로젝트는 부모님이 자녀의 독립 자금을 시뮬레이션해 보는 웹 애플리케이션입니다. 현재 v2.2.0까지 개발되었으며, 아래의 기획 의도와 규칙을 숙지하고 이어서 개발을 진행해 주세요.

## 1. 프로젝트 개요 및 컨셉
* **서비스명:** 아빠의 '서른 살엔 독립해라' 계산기
* **핵심 타겟:** 자녀에게 목돈을 모아주고 싶고, 독립할 때 든든한 자금을 마련해 주고 싶은 부모님들.
* **톤앤매너:** '전문적이고 따뜻한 부모의 마음'. 신뢰할 수 있는 금융 시뮬레이터의 정확성을 갖추되, 안내 문구와 결과 텍스트는 부모의 따뜻한 마음이 느껴지는 편안하고 전문적인 어조로 작성합니다. 과도한 유머 표현이나 속어는 지양하고, 이모지는 정보 전달을 돕는 경우에 한해 절제하여 사용합니다.

## 2. 기술 스택 및 아키텍처
* **프론트엔드 환경:** 바닐라(Vanilla) HTML5, CSS3, ES6 JavaScript (프레임워크 미사용)
* **파일 구조 (분리 완료):** `index.html` (구조), `style.css` (스타일), `script.js` (동작)
* **외부 라이브러리:**
  * 차트: `Chart.js` (CDN 방식)
  * 폰트: `Pretendard` (CDN 방식)
* **UI/UX 디자인:** Material Design 감성 적용. 부드러운 그림자(Box-shadow), 둥근 모서리(Border-radius 8~16px), 입력창 포커스 애니메이션 등 깔끔하고 신뢰감 있는 UI. 데스크탑 2분할(좌측 입력, 우측 결과) / 모바일 1단 반응형 구조.

## 4. 핵심 비즈니스 로직 (Core Logic)
* **자산 성장 (복리 계산):**
  * 거치식 복리: 기존에 증여한 '씨드머니'가 남은 기간 동안 굴러가는 가치
  * 적립식 복리: 매월 추가로 납입하는 금액이 굴러가는 가치
  * 총합: 두 복리의 합산을 최종 자금으로 산출
* **목표 가치 (인플레이션 계산):**
  * 사용자가 선택한 선물(학자금, 주택마련 등)의 현재 가치에 연간 물가상승률(3%)을 적용해 목표 시점의 실제 필요 금액을 산출
* **증여세 한도 (납입한도 최대 적용 기준):**
  * 미성년자: 10년 합산 2,000만원 한도 → 월 최대 납입액 자동 계산
  * 성인(만 19세 이상): 10년 합산 5,000만원 한도 → 월 최대 납입액 자동 계산
  * 미성년 + 성인 혼합 구간: 각 구간 독립 계산 후 합산 적용

## 5. 주요 UI 구성요소
* **월 납입 모드 셀렉터 (v2.2.0 기준):**
  * `직접 입력`: 사용자가 월 납입액을 직접 입력
  * `납입한도 최대 적용`: 증여세 한도 기준으로 최대 월 납입액을 자동 계산해 적용. 미성년/성인/혼합 구간 모두 지원
  * 기존의 '성년 후 최대 납입 자동 전환' 토글은 이 셀렉터로 통합·대체됨
* **씨드머니:** 시작 시점에 증여하는 초기 목돈
* **목표 선물 카드:** 학자금, 주택 전세/매매 등 선물 유형을 카드 UI로 선택
* **차트 + 실시간 요약 영역:** Chart.js 기반 복리 성장 시각화, 차트 외부 요약 텍스트(모바일 가시성 대응)
* **증여세 타임라인:** 납입 스케줄과 증여세 면제 한도를 시각적으로 안내
