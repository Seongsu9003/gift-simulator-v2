# GEMINI.md - 아빠의 '서른 살엔 독립해라' 계산기 (Gift Simulator)

이 문서는 **자녀 증여 시뮬레이터** 프로젝트에서 작업하는 AI 에이전트를 위한 지침과 프로젝트 맥락을 제공합니다.

## 프로젝트 개요

*   **목적:** 한국의 부모님들이 자녀를 위한 장기 증여 전략을 세울 수 있도록 돕는 금융 도구입니다. 복리 수익을 통한 자산 증식과 물가 상승률을 반영한 목표 금액을 비교하며, 한국 증여세법에 따른 비과세 혜택 타임라인을 제공합니다.
*   **주요 타겟:** 자녀의 경제적 독립(예: 30세)을 체계적으로 준비하려는 부모님.
*   **핵심 로직:**
    *   **복리 계산:** 거치식(씨드머니)과 적립식(매월 납입금)을 합산하여 계산합니다. (기본 수익률: S&P 500 기준 10.5%)
    *   **물가 상승률:** 목표 선물(학자금, 주택 자금 등)의 현재 가치를 미래 가치로 변환합니다. (기본 물가상승률: 3.5%)
    *   **증여세 타임라인:** 10년 주기 비과세 한도(미성년 2천만 원, 성인 5천만 원)를 시각화합니다.
    *   **유기정기금 평가:** 정기적인 증여 시 3% 할인율을 적용한 비과세 한도 내 최대 월 납입액을 계산합니다.

## 기술 스택

*   **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
*   **Typography:** [Pretendard](https://github.com/orioncactus/pretendard) (웹 폰트).
*   **데이터 시각화:** [Chart.js](https://www.chartjs.org/) (CDN 사용).
*   **이미지 내보내기:** [html2canvas](https://html2canvas.hertzen.com/) (CDN 사용).
*   **배포:** Vercel.

## 프로젝트 구조

*   `index.html`: 메인 입력창, 결과 대시보드 및 이미지 생성을 위한 숨겨진 템플릿 포함.
*   `style.css`: 레이아웃, 반응형 디자인, 타임라인 UI 스타일링.
*   `script.js`: 핵심 로직 포함.
    *   한글 금액 표기 유틸리티 (`formatKoreanAmount`, `formatApproximateCurrency`).
    *   복리 및 물가 상승률 계산.
    *   증여세 마일스톤 생성 로직.
    *   Chart.js 차트 초기화 및 업데이트.
    *   `html2canvas`를 이용한 증여 계획서 이미지 다운로드 기능.
*   `README.md`: 프로젝트 소개 및 주요 기능 설명.

## 개발 가이드라인

### 실행 방법
*   **로컬 개발:** 별도의 빌드 과정이 필요 없습니다. `index.html`을 브라우저에서 직접 열거나 VS Code의 Live Server 등을 사용하여 실행하세요.
*   **의존성:** `Chart.js`, `html2canvas`는 CDN을 통해 로드됩니다.

### 코딩 표준
*   **Vanilla JS:** 프로젝트의 경량성을 유지하기 위해 프레임워크 없이 순수 자바스크립트 사용을 권장합니다.
*   **수치 계산:** 금융 계산은 최대한 정밀하게 수행한 후, 출력 시에만 `Math.round()` 또는 `Math.floor()`를 사용하여 정리합니다.
*   **현지화:** 모든 UI와 금액 표기는 한글 및 원화(KRW) 기준을 유지합니다.

### 기능 추가 시 주의사항
*   새로운 금융 변수를 추가할 경우, `script.js`의 `calculate()` 함수와 차트 업데이트 로직을 함께 수정해야 합니다.
*   UI 수정 시, 결과 이미지(`captureTemplate`)에도 변경 사항이 반영되어야 하는지 확인하세요.

## 주요 상수 및 가정
*   **기본 수익률:** 10.5% (사용자 수정 가능).
*   **기본 물가상승률:** 3.5% (사용자 수정 가능).
*   **유기정기금 할인율:** 3% (국세청 기준 고정).
*   **비과세 한도:** 미성년 2,000만 원, 성인 5,000만 원.
